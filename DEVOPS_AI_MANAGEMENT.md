## ⚡️ GitHub Infrastructure

### 1. Repository Access
The source code is hosted at: `https://github.com/Tehuti01/thunderforge-vst.git`
- **Primary Branch**: `main`
- **Submodules**: Includes `NeuralAmpModelerCore` for the NAM DSP engine.

### 2. Automated Scripts (The "One-Command" Flow)
